Data Instructions for Atlantis!

For your data section, upload a CSV with the following columns:

1. Sample1, Sample2, Sample3 ... etc. (Each Sample must be labelled in the format Sample'i').
2. gene which denotes the gene name.
3. log2f which denotes the log fold change.
4. se which denotes the standard error.
5. p_values which denotes the p_values for the differential expression of that gene.

Data Annotation Instructions for Atlantis!

For your data section, upload a CSV with the two columns:

1. Sample: This column should contain the name of the samples in order. Eg. Sample1, Sample2, Sample3, Sample4 etc.
2. Condition: This column should contain the numerical indication of the condition that the sample was placed in. Eg. 1, 2, 3 etc.
